# IPA図表・表付き問題の追加テンプレート

AP Study v2.7.0 では、既存の `Question` データを変更せず、任意の `assets` と `tables` で図表を追加できます。問題文・選択肢・正解・図表は、必ずIPA公式問題PDFと公式解答で照合してください。

画像付き問題を追加する場合は、まず [手動図表画像投入ワークフロー](./ipa-manual-figure-asset-workflow.md) に従って画像を `public/assets/ipa/ap/...` に配置し、`npm run verify:questions` で画像パス、実ファイル、サイズ、拡張子、`alt` を確認してください。候補の状態は [図表問題候補リスト](./ipa-figure-question-candidates.md) で管理します。

## 画像付き問題のデータ例

以下は記述形式を示すテンプレートです。問題番号、本文、選択肢、正解、解説、URLを推測して投入してはいけません。

```ts
{
  id: 'ap-r06-spring-am-q023',
  examYear: 2024,
  examSeason: '春期',
  examType: 'morning',
  questionNumber: 23,
  field: FIELDS.network,
  subField: 'ネットワーク構成',
  questionText: '次の図に示すネットワーク構成に関する記述として、適切なものはどれか。',
  assets: [
    {
      type: 'image',
      src: '/IPA_AP_STUDY/assets/ipa/ap/r06-spring/am/q023-figure-1.webp',
      alt: '令和6年度 春期 午前 問23のネットワーク構成図',
      caption: 'ネットワーク構成図',
      sourceName: '情報処理推進機構（IPA） 応用情報技術者試験 令和6年度 春期 午前 問23',
      sourceUrl: '公式PDFの確認済みURL',
    },
  ],
  choices: [
    { key: 'ア', text: '...' },
    { key: 'イ', text: '...' },
    { key: 'ウ', text: '...' },
    { key: 'エ', text: '...' },
  ],
  correctAnswer: 'ア',
  officialAnswerText: 'ア：...',
  sourceName: '情報処理推進機構（IPA） 応用情報技術者試験 令和6年度 春期 午前 問23',
  sourceUrl: '公式PDFの確認済みURL',
  isQuoteFromIpa: true,
  explanation: {
    correctReason: '...',
    wrongReasons: { ア: '...', イ: '...', ウ: '...', エ: '...' },
    points: ['...'],
    keywords: ['...'],
    isAiGenerated: false,
  },
}
```

## 表付き問題のデータ例

表は画像化せず正確に転記できる場合、`headers` と `rows` を使います。各行の列数は `headers` と一致させます。

```ts
{
  // 通常のQuestionフィールドは省略
  tables: [
    {
      caption: '処理時間の測定結果',
      headers: ['処理', '1回目', '2回目'],
      rows: [
        ['処理A', '10ミリ秒', '12ミリ秒'],
        ['処理B', '20ミリ秒', '18ミリ秒'],
      ],
      sourceName: '情報処理推進機構（IPA） 応用情報技術者試験 令和○年度 ○期 午前 問○',
      sourceUrl: '公式PDFの確認済みURL',
    },
  ],
}
```

画像と構造化表の両方が必要な場合は、同じ問題に `assets` と `tables` を併記できます。

## assets配置ルール

- 配置先は `public/assets/ipa/ap/{era-season}/am/` とする。
- 公開パスは `/IPA_AP_STUDY/assets/ipa/ap/{era-season}/am/...` とする。
- 例：`public/assets/ipa/ap/r06-spring/am/q023-figure-1.webp`
- 午前問題だけを対象とし、今回の基盤を午後問題の投入には使用しない。
- WebPを推奨し、1枚100〜300KB程度を目安にする。
- TypeScriptへbase64で埋め込まない。公式PDFを丸ごと配置しない。
- 図表の周囲に問題文、ページ番号、別問題の内容が残らないよう、必要範囲だけを正確に切り出す。

## 画像ファイル命名ルール

- 図：`qNNN-figure-1.webp`
- 同じ問題の2枚目以降：`qNNN-figure-2.webp`、`qNNN-figure-3.webp`
- 表を画像として保持する場合：`qNNN-table-1.webp`
- `NNN` は午前問題の3桁問番号とする。
- 年度・期はディレクトリ名（例：`r06-spring`、`r05-autumn`）で表現する。

## alt / caption / sourceName / sourceUrl

- `alt`（必須）：画像を見られない利用者が設問を理解できるよう、年度・期・問番号と図の種類を簡潔かつ具体的に書く。ただし図から導くべき正解を説明しない。
- `caption`（任意）：画面上で図表の意味を短く示す。公式資料に表題がある場合は、その表記を優先する。
- `sourceName`（任意）：`情報処理推進機構（IPA） 応用情報技術者試験 令和○年度 ○期 午前 問○` の形式を基本とする。
- `sourceUrl`（任意）：実際に照合したIPA公式ページ又は公式問題PDFのHTTPS URLだけを設定する。不明な場合は推測せず省略する。

## 図表問題追加時のチェックリスト

- [ ] IPA公式問題PDFで問題文、選択肢、図表、問番号を照合した。
- [ ] IPA公式解答で正解を照合した。
- [ ] 既存問題IDと重複せず、既存IDを変更していない。
- [ ] 切り出した画像に欠落、変形、文字化け、不要な別問題がない。
- [ ] `asset.type` は `image`、`src` と `alt` は空でない。
- [ ] `src` は `/IPA_AP_STUDY/assets/ipa/ap/` で始まり、対応ファイルが `public/assets/ipa/ap/` に存在する。
- [ ] 表の `headers` と `rows` は空でなく、全行の列数が一致する。
- [ ] 出典名とURLは確認済みであり、推測を含まない。
- [ ] `npm run verify:questions`、`npm run build`、`npm run verify:dist` が成功する。

## スマホ表示確認項目

- [ ] 画像が画面幅から横にはみ出さない。
- [ ] 画像タップと「拡大して見る」の両方でモーダルを開ける。
- [ ] 閉じるボタン、背景タップ、Escキーでモーダルを閉じられる。
- [ ] モーダル中にページが横スクロールせず、固定ヘッダー／フッターより前面に出る。
- [ ] 表は文字が読める大きさを保ち、必要な場合だけ表領域を横スクロールできる。
- [ ] ライトテーマとダークテーマで、本文、罫線、見出し、出典リンクを判読できる。
- [ ] 通常演習、問題詳細、午前模試、問題一覧、検索から同じ問題を開いても表示が崩れない。
- [ ] 問題一覧と演習画面に「図表あり」又は「表あり」ラベルが表示される。

## 図表問題をスキップすべき条件

次のいずれかに該当する場合は、データを推測で補わず「図表対応待ち」又は「図表切り出し未対応」として管理します。

- 公式問題PDF又は公式解答を確認できない。
- 図表の切り出し範囲、問番号との対応、ページ構成が曖昧である。
- 抽出画像の文字が小さすぎる、欠ける、潰れる、又はPDF抽出で配置が崩れる。
- 色、線種、凡例、注記など、解答に必要な情報を忠実に再現できない。
- 問題文・選択肢・正解・出典URLのいずれかを推測する必要がある。
- 適切なaltを、正解のネタバレなしに記述できない。
- 画像が過大で、WebP化や適切な切り出しをしても実用的な容量・可読性にならない。

## 取り込み状況の表現

PR本文又は過去問カタログの注記では、年度・期ごとに次の状態を区別します。

- **テキスト問題として取り込み済み**：図表なしで正確に成立する問題を投入済み。
- **図表対応待ち**：問題は確認済みだが、表示基盤又はデータ整備待ち。
- **図表付きで取り込み済み**：公式確認済みの図表ファイルと問題データを投入済み。
- **図表切り出し未対応**：公式PDFは確認できるが、品質を満たす画像をまだ作成していない。

## 図表問題候補の管理

候補問題は [図表問題候補リスト](./ipa-figure-question-candidates.md) に独立して記録します。画像ファイルが未配置、又は公式資料で内容を確認できない候補は、問題データへ追加しません。
