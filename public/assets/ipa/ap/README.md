# IPA AP午前問題の図表画像

このフォルダは、IPA応用情報技術者試験の午前問題で必要となる図表画像を置く場所です。

- IPA公式PDFを丸ごと配置しないでください。
- 対象問題に必要な図表部分だけを、欠けや不要な周辺情報がないように切り出してください。
- WebPを推奨します。検証で許可される形式はWebP、PNG、JPG、JPEG、SVGです。
- 配置とファイル名は `{era-season}/am/qNNN-figure-1.webp` とします。例: `r04-autumn/am/q052-figure-1.webp`。
- 同じ問題の2枚目以降は `qNNN-figure-2.webp`、`qNNN-figure-3.webp` と連番にします。
- 500KB以下を目安とします。500KB超は検証警告、1MB超は検証エラーです。
- 可読性を損なう圧縮、推測による描き直し、AI生成画像、第三者サイト画像の代用は禁止です。
- `sourceUrl`、`sourceName`、`alt`、必要に応じた `caption` は画像ファイルではなく問題データ側の `assets` に持たせます。
- 画像ファイルだけを配置しても出典表示は完結しません。公式資料で問題内容と正解を照合し、問題データに確認済みの出典情報を設定してください。

詳細は [`docs/ipa-manual-figure-asset-workflow.md`](../../../../docs/ipa-manual-figure-asset-workflow.md) を参照してください。
